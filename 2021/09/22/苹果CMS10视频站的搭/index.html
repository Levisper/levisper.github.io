苹果CMS10搭建视频站

2021-09-22
字数统计: 701   |   阅读时长≈ 2 分钟

苹果CMS程序是一套采用PHP+MYSQL环境下运行的完善而强大的快速建站系统。因此，通过它，我们可以方便快捷地搭建个人视频站点。

一：准备工作

1、VPS一台

推荐购买国外商家，选择配置较高点的

2、宝塔环境安装

宝塔官网

安装教程

环境配置：Nginx + PHP (5.6 +) + MySQL(5.5 +)

3、域名解析

如果没有域名，先去购买一个，国内外域名服务商均可

解析你购买的域名到用于部署图床的VPS,A类型解析 (推荐CloudFlare平台)

4、CMS程序下载

GitHub

CMS官网

下载程序压缩包至本地

二：网站部署

1、宝塔中添加站点

登陆宝塔之后，点击网站—添加站点

填如相应的域名，选择数据库并更改账号和密码，勾选php版本

2、上传CMS程序至网站根目录

点击网站根目录，删除目录下的 index.html 和 404.html 文件

上传前面下载好的CMS压缩包

解压文件，并将文件夹里的内容全选之后剪切，粘贴到根目录下，如图所示

给网站申请免费ssl证书，并开启强制HTTPS

3、配置网站

在浏览器中输入你的网址，会自动跳转到 https://你的域名/install.php，并自动运行 install.php 脚本进行安装初始化

初始化后，fileinfo 处于红色状态，此时需要在宝塔中安装此扩展

点击软件商店—已安装—PHP设置

安装fileinfo扩展

安装完成后，重载配置，回到网站界面并刷新，可以发现红色标记已经消失

填入数据库名称、账号及密码（可在宝塔面板的数据库一栏查看），默认覆盖数据库，并测试数据库连接是否正确，自行设置管理员账号和密码（请保存好，以便后续登陆），勾选创建初始化数据，完成以上设置后（未说明的不需要更改），点击立即执行安装

4、宝塔中重命名网站根目录的admin.php文件

可以修改成任意名字，注意**.php**后缀不变

5、登陆后台

完成第四步后，刷新就可以登陆进后台，如提示更新程序，更新即可

6、采集资源

点击采集—自定义接口—添加

自行谷歌影视资源采集站，并按提示进行填写

详尽配置可以参考CMS对接视频库

7、模板的使用

可以自行套用模板：模板参考

替换教程模板里会有对用的教程提示
